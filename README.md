# Курсова робота на тему: Розробка та реалізація інформаційної системи для медичного закладу з базою даних пацієнтів.

# Ідея: Веб-застосунок для управління лікарнею з трьома основними модулями: Реєстратура, Лікар та Головний лікар.

## Технології

- **Backend**: Python Flask
- **Frontend**: HTML, CSS, JavaScript
- **База даних**: MySQL

## Встановлення

### 1. Встановіть залежності

```bash
pip install -r requirements.txt
```

### 2. Налаштуйте базу даних

1. Імпортуйте базу даних з файлу `DB/hospital_db.sql`:

2. Відкрийте файл `app.py` та налаштуйте підключення до бази даних:
```python
DB_CONFIG = {
    'host': 'localhost',
    'database': 'hospital_db',
    'user': 'root',
    'password': '2510',  
    'charset': 'utf8mb4',
    'collation': 'utf8mb4_unicode_ci'
}
```

### 3. Запустіть додаток

```bash
python app.py
```

Додаток буде доступний за адресою: http://localhost:5000

## Функціональність

### Стартова сторінка
- Три великі кнопки для переходу до різних модулів

### Реєстратура
- Перегляд всіх пацієнтів
- Пошук пацієнтів по прізвищу
- Додавання нового пацієнта 
- Запис пацієнта на візит 

### Лікар
- Вибір лікаря зі списку
- Перегляд візитів на сьогодні, минулих майбутніх та усіх візитів
- Відкриття картки прийому
- Перегляд історії діагнозів пацієнта
- Додавання нового діагнозу та завершення прийому

### Головний лікар (Аналітика)
- загальна статистика (пацієнти, візити за місяць, лікарі)
- Топ-5 лікарів за кількістю візитів
- Топ діагнозів з відсотками
